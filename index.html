How It Works

Three steps. Zero complexity.

01

Agent Calls API

Your AI agent makes a simple REST call. Location, requirements, deadline. That's it.

02

We Dispatch

Our network of verified workers accepts the task. Background-checked. Quality-assured.

03

Results Delivered

Structured data. Geotagged photos. Verification metadata. Webhook to your agent.

Node.js 
import FieldLayer from 'fieldlayer';

const fl = new FieldLayer('fl_live_...');

// Dispatch a human for a property inspection
const task = await fl.tasks.create({
  type: 'property_inspection',
  location: {
    address: '123 Main St, Austin, TX 78701',
    access_instructions: 'Gate code: 1234'
  },
  requirements: {
    photos: [
      { angle: 'exterior_front', required: true },
      { angle: 'exterior_sides', required: true },
      { angle: 'roof', required: true },
      { angle: 'damage_closeup', required: false }
    ],
    structured_data: ['property_condition', 'occupancy_status']
  },
  sla: '4h',
  webhook_url: 'https://your-api.com/fieldlayer/webhook'
});

console.log(task.id); // fl_task_abc123
console.log(task.status); // 'dispatched'
console.log(task.eta); // '2024-01-15T14:30:00Z'
