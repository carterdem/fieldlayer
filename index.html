How It Works

Three steps. That's it.

Step 1

Your Agent Calls the API

Simple REST call with task details — location, requirements, deadline. That's all your agent needs.

Step 2

We Dispatch a Human

Our network of verified workers accepts the task. Background-checked, trained, ready to go.

Step 3

Results Delivered

Structured data, geotagged photos, and verification — delivered via webhook, ready for your agent.

Simple, Powerful API

One call to dispatch. One webhook for results.

create-task.js 
const task = await fieldlayer.tasks.create({
  type: "property_inspection",
  location: {
    address: "123 Main St, Austin, TX"
  },
  requirements: {
    photos: ["exterior_front", "exterior_sides", "roof"],
    data: ["property_condition", "occupancy_status"]
  },
  deadline: "4h"
});

// Webhook delivers results when complete
